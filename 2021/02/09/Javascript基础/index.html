Javascript基础语法

变量

一、数据类型

1. Number

  • 八进制：数字前加0

  • 十六进制：数字前加 0x

  • 最大值：Number.MAX_VALUE

  • 最小值：Number.MIN_VALUE

  • 无穷大：Infinity

  • 非数字：NaN

  • isNaN：判断是否为非数字类型，若为数字返回false

2. String

  • 字符串需要包括在单引号后者双引号之中

  • 转义字符：\n、\t、\b等

  • 计算字符串长度，使用属性：legnth

  • 字符串连接使用+

3. Boolean

  • true和数字运算时结果为1，false和数字运算结果为0

4.Undefined

  • 未定义数据类型与数字运算结果为NaN

5. Null

  • 空类型与数字运算为0

二、typeof

  • 获取数据的类型

三、数据类型转换

1. 转化为字符串类型

  • toString()

  • String()

  • 使用“+”拼接

2. 转化为数字类型

  • parseInt()，会舍去小数部分，可以去除数值后面的单位

  • parseFloat()，可以去除数值后面的单位

  • Number()

  • 进行减乘除运算

3. 转化为布尔类型

  • Boolean()，其中” “、0、null、undefined、NaN转化为false，其余转化为true

运算符

一、算数运算符

  • 加（+）、减（-）、乘（*）、除（/）、取余（%）

二、自运算

  • 前置自增（自减），先运算，后返回值。
  • 后置自增（自减），先返回值，后运算。

三、比较运算符

  • 大于（>）、小于（<）、大于等于（>=）、小于等于（<=）、等于（==），不等于（!=）、全等于（===），不全等（!==）
  • 大于（>）、小于（<）、大于等于（>=）、小于等于（<=）、等于（==），
    存在隐式转换、全等（===），必须是值和数据类型完全相同。

四、逻辑运算符

  • 逻辑与（&&）
    • 全部为true，则结果为true
  • 逻辑或（||）
    • 只要有一个结果为true，结果为true
  • 逻辑非（!）
    • 取反，true取反为flase；false取反为true
  • 逻辑中断
    • 多个表达式运算时，当获取到确定结果时 ，返回结果，后续代码将不再运行。

五、赋值运算符

  • 赋值（=）、加等于（+=）、减等于（-=）、乘等于（*=）、除等于（/=）、取余等于（%=）

六、运算符优先级

  1. 括号 （）
  2. 单元运算符（++、–、！）
  3. 算数运算符（*、/、%、+、-，先乘除后加减）
  4. 比较运算符（>=、<=、>、<、==、===、!=、!==）
  5. 逻辑运算符（&&、||，先&&后||）
  6. 赋值运算符（=）
  7. 逗号运算符 （，）

流程控制

一、顺序

  • 程序依据顺序执行

二、分支

1. if（条件表达式）{…}-else{…}

  • 条件表达式为true执行if里面代码块，否则执行else里面代码块

2.三元表达式

  • 条件表达式？表达式一：表达式二；条件表达式结果为true执行表达式一，否则执行表达式二

3. switch

  • case，匹配switch的条件表达式
  • break，退出switch
  • default，默认选项，若没有匹配到任何case，执行此语句

三、循环

1. for（var i=0；i<number；i++）{…}

  • 最常用，经常使用嵌套循环

2.while(表达式){…}

  • 先判断再执行循环体

3、do{…}while(表达式)

  • 先执行循环体再判断

4、关键字

  • continue，跳出当前循环，继续执行下一次循环
  • break，退出整个循环

函数

一、概述

  • 一段可以重复使用的代码块

二、函数声明

三、参数

  • 形参
    • 形式上的参数，函数声明时内部使用
  • 实参
    • 实际参数，函数调用时赋值给形参带入函数体使用
  • 形参多于实参
    • 多余的形参值为undefined，运算结果为NaN
  • 实参多于形参
    • 按照实际的形参个数进行执行
  • 默认参数
    • 可以直接给形参默认值，当输入实参时覆盖默认值
  • 无法确定参数个数
    • 使用arguments接收传递来的实参，arguments是一个伪数组，具有length属性，可以按照索引取值，可以遍历，但是不具备数组的一些方法

四、返回值

  • return，函数只有一个返回值，当函数返回时后面代码不再执行

作用域

一、全局作用域

  • 全局作用域定义的变量为全局变量，变量不声明直接赋值也可以直接转化为全局变量

二、局部作用域

  • 局部作用域定义的变量为局部作用域，局部变量，在代码块中声明的变量

三、注意

  • 在ES5中没有块级作用域

  3. 代码执行
    • 代码按照顺序执行
